Formatações adicionais no HTML5

Texto marcado

Podemos criar um texto marcado usando a tag "mark".

Outro texto marcado

Textos marcados de uma cor só

Crie uma tag style, e dentro dela selecione o mark.
(Deixei as bordas redondas usando border-radius).

Texto grande e pequeno

A tag big está obsoleta, por quê é uma tag sem sentido.
Pode-se usar a tag small para deixar uma palavra pequena.

Texto deletado

Podemos marcar um texto como excluído para indicar
que ele pode ser lido mas não considerado. (Use a tag del)

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase
e indicar que ele foi adicionado depois. (Use a tag ins)

Pode-se usar a tag U (não semântica)

Texto sobrescrito

Para inserir coisas do tipo x20+3 (use a tag sup) Sup: Deixa em cima.

Texto subscrito

Para inserir coisas do tipo H2O (Use a tag sub) Sub: Deixa em baixo.
